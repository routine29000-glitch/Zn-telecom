# ZN Ooredoo

تطبيق Android (Kotlin) لتفعيل عروض أوريدو (الهدية اليومية + عروض الإنترنت
الأسبوعية) عبر أتمتة USSD، مع إعلانات AdMob.

## لماذا Ooredoo فقط؟

Djezzy تتطلب أتمتة تطبيقهم الرسمي (accessibility tap automation)، وهو حل
أهش ويحتاج صيانة مستمرة كل ما يحدثو تصميم تطبيقهم. Ooredoo تشتغل بكود USSD
حقيقي وثابت (`#500#`)، فهذا أبسط وأكثر استقراراً بكثير، وهو ما ركّزنا عليه
في هذا الإصدار.

## الحالة الحالية

- شاشة Splash بالشعار
- شاشة اختيار نوع العملية: الهدية اليومية / عروض الإنترنت الأسبوعية + Banner Ad
- شاشة رقم الهاتف
- شاشة رمز التحقق (OTP) — تُقرأ تلقائياً من SMS الحقيقي الذي يرسله Ooredoo
  (عبر `SmsCodeReceiver`)
- شاشة التفعيل (Interstitial Ad بعد النجاح + Rewarded Ad اختياري)
- خدمة Accessibility (`OperatorAccessibilityService`) تقرأ قائمة USSD
  `#500#` وتضغط أوتوماتيكياً على الخيار المناسب (حالياً: الخيار "5"
  للهدية اليومية)

## ما ينقص قبل الاستخدام الفعلي (TODO)

### تأكيد رقم الخيار الحقيقي في قائمة USSD

افتح:
```
app/src/main/java/com/zntelecom/app/service/OperatorAccessibilityService.kt
```
القيمة الحالية `DAILY_GIFT_OPTION = "5"` مبنية على معلومة عامة، وقد تختلف
باختلاف حسابك/منطقتك. اتصل بـ `#500#` يدوياً مرة واحدة وشاهد ترقيم القائمة
الحقيقي عندك، وعدّل القيمة إذا لزم الأمر. لعروض الإنترنت الأسبوعية، أضف
منطق مشابه لاختيار الخيار الصحيح داخل نفس الدالة `handleUssdDialog()`.

### دعم أجهزة مختلفة (dialer package)

اسم حزمة تطبيق الهاتف/USSD dialog يختلف أحياناً حسب الشركة المصنعة
(Samsung/Xiaomi/etc تستعمل أحياناً حزمة مختلفة عن `com.android.phone`).
إذا لاحظت أن الأتمتة ما تشتغلش على جهاز معين، أضف اسم الحزمة الصحيح في:
- `app/src/main/res/xml/accessibility_service_config.xml` (قيمة `packageNames`)
- `OperatorAccessibilityService.kt` (قيمة `PHONE_PACKAGE`)

## البناء عبر GitHub Actions (مجاني)

1. أنشئ مستودع جديد على GitHub وارفع محتوى هذا المجلد كامل إليه
2. بمجرد الرفع (push) على فرع `main`، سيبدأ البناء تلقائياً
3. من تبويب "Actions" في المستودع، افتح آخر عملية بناء، وحمّل
   الملف من قسم "Artifacts" باسم `zn-telecom-debug-apk`
4. ثبّت الـ APK يدوياً على هاتفك (سيطلب منك تفعيل "تثبيت من مصادر غير معروفة")

## خطوات المستخدم النهائي (مرة واحدة فقط)

1. تثبيت الـ APK
2. تفعيل خدمة Accessibility: الإعدادات ← تسهيلات الاستخدام ← ZN Ooredoo ← تفعيل
3. الموافقة على صلاحيات SMS والاتصال عند أول طلب
